## Collect. Curate. Embed.

Praised is a full-stack SaaS platform for managing customer testimonials. Create a branded space, share a collection link, review submissions in a dashboard, and embed a one-line widget on any website.

### Features

- Branded collection forms with customizable appearance
- Approval workflow — review before anything goes live
- Embeddable "Wall of Love" widget (one script tag)
- Stripe subscription billing (free → $49/mo)
- 40 Jest unit tests

### Stack

TypeScript · Express.js · SQLite (WAL mode) · JWT auth · Stripe · Docker
